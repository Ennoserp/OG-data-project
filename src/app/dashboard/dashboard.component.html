Welcome to Olympic Games Data !
The website that provides you information about the Olympic Games.

This website shows the number of medals won by countries and by disciplines. You can click on Map to see the world maap. It shows you the number of medals won by countries. List of Countries shows you the list of all the countries and you can click on any of them to see more details. List of Disciplines shows you the list of all the disciplines during the Olympic Games. List of Games displays (should) a list of all Olympic games edition.
